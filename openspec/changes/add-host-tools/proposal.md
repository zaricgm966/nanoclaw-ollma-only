# 变更提案：补回宿主机操作工具链

## 背景

当前 NanoClaw 已经切换为 Ollama-only runner，并保留了联网搜索能力，但原先在记忆文档中提到的浏览器自动化、截图、桌面操作等能力没有实际执行链。结果是 agent 仍会口头声称自己可以打开应用或截图，但实际只能调用 `web_search` 和 `web_fetch`。

## 目标

- 为当前 NanoClaw 补回最小可用的宿主机操作能力
- 首批支持 `open_app` 与 `take_screenshot`
- 修正文档和记忆中与真实能力不一致的描述
- 保持现有 Web 搜索与直聊链路不受影响

## 非目标

- 不在本次变更中实现完整桌面自动化
- 不在本次变更中实现宿主机任意命令执行
- 不在本次变更中实现 Web UI 图片附件预览
